# 📱 Termux Text-To-Speech (gTTS Pro)

টার্মাক্স (Termux)-এর জন্য একটি সহজ এবং কার্যকরী টেক্সট-টু-স্পিচ (TTS) স্ক্রিপ্ট। এটি গুগলের অফিশিয়াল TTS ইঞ্জিন ব্যবহার করে যেকোনো বাংলা টেক্সটকে নিখুঁত অ디오তে (MP3) রূপান্তর করে এবং সরাসরি ফোনের ইন্টারনাল স্টোরেজের **Download** ফোল্ডারে সেভ করে।

✨ বৈশিষ্ট্যসমূহ (Features)
​Natural Bengali Voice: গুগলের ক্লাউড বেসড ভয়েস ব্যবহার করায় উচ্চারণ একদম স্পষ্ট ও স্বাভাবিক শোনায়।
​Storage Integration: ফাইলটি টার্মাক্সের ভেতরে লুকিয়ে না থেকে সরাসরি ফোনের Download ফোল্ডারে সেভ হয়।
​User Friendly: নতুনদের জন্য ব্যবহার করা খুবই সহজ এবং কোনো জটিল কনফিগারেশন নেই।
​🛠️ প্রয়োজনীয় প্যাকেজ ইনস্টলেশন (Installation)
​যারা এই টুলটি ব্যবহার করতে চান, তারা প্রথমে আপনার টার্মাক্স অ্যাপটি ওপেন করুন এবং নিচের কমান্ডগুলো একে একে কপি করে রান করুন:

# ১. টার্মাক্স রিপোজিটরি আপডেট করুন
pkg update && pkg upgrade -y

# ২. পাইথন এবং গিট ইনস্টল করুন
pkg install python git -y

# ৩. গুগলের টেক্সট-টু-স্পিচ লাইব্রেরি ইনস্টল করুন
pip install gTTS

# ৪. টার্মাক্সকে ফোনের মেমোরি ব্যবহারের পারমিশন দিন
termux-setup-storage

(নোট: শেষ কমান্ডটি দেওয়ার পর ফোনে একটি পপ-আপ আসবে, সেখানে অবশ্যই Allow বা পারমিশন দিতে হবে।)
​🚀 যেভাবে ব্যবহার করবেন (How to Run)
​সব প্যাকেজ ইনস্টল হয়ে গেলে টুলটি ক্লোন করে রান করার জন্য নিচের কমান্ডগুলো দিন:

# ১. রিপোজিটরিটি ক্লোন করুন
git clone https://github.com/RanaCoding-cs/--.git

# ২. টুলস ফোল্ডারে প্রবেশ করুন
cd YOUR-REPO-NAME

# ৩. স্ক্রিপ্টটি রান করুন
python tts_pro.py

⚠️ জরুরি নোট (Important Note)
​ইন্টারনেট কানেকশন: এই স্ক্রিপ্টটি কাজ করার জন্য আপনার ফোনে অবশ্যই ইন্টারনেট চালু থাকতে হবে, কারণ এটি গুগলের অনলাইন সার্ভার ব্যবহার করে অ디오 তৈরি করে।
​পারমিশন এরর: যদি ফাইল সেভ করার সময় Permission Denied দেখায়, তবে নিশ্চিত হয়ে নিন যে আপনি termux-setup-storage কমান্ডটি দিয়ে ফোনে স্টোরেজ পারমিশন সঠিকভাবে দিয়েছেন।
